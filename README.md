# MAX-Telegram
Проект для пересылки сообщений из MAX в Telegram

## Как Настроить:
1. Переходим на сайт https://green-api.com и регистрируемся
2. Создаём инстанс для MAX
3. Через qr в green api входим в MAX
4. Во вкладке чаты находим чат и берём id
5. Пишем id в код в payload
6. Переходим во вкладку (Все методы...) и находим GetChatHistory
7. Из кода в GetChatHistory берём url и вставляем в код Python
8. В target_sender пишем отправилеля из MAX
9. Создаём бота в Telegram через Bot Father, добавляем в чат и вставляем токен в код
10. Указываем в chat_id_tg id чата в telegram
